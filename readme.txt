About the Application
1. Source node highlighted in red
2. Destination node highlighted in blue
3. Color is given to each link and text depending on the type of carrier.
4. Hovering over the nodes shows the number of edges leaving and entering the particular node.

Steps
1. Choose departure airport
2. Showes carriers and destinations for the departure airport.
3. Can choose multiple carriers and destinations.
4. Below the graph shows the detailed flight information.
